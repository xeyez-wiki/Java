# Java
Java 7, 8 Examples

## tryWithResources
Java 7부터 지원.
AutoCloseable Interface 구현이 된 Class(Resource 관련 Class)의 경우 <br />
"close()"가 자동으로 호출되어 간결한 표현 가능. <br />
(이용하는 쪽에서 "close()" 명시적으로 서술 필요 없음.)
